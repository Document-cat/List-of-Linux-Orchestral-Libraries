# List of Linux Native Orchestral Libraries
I was inpired by this post https://linuxmusicians.com/viewtopic.php?t=29267 in the LinuxMusicians forum to create my own list of Linux Native Orchestral Libraries.

This list is incomplete any help in making it more complete is apreciated.
